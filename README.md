# Dashboard-Project - Karlsruhe-Strasbourg

## Installation

On macOS and Linux:

```sh
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

On Windows:

```sh
py -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
```

To run:
```sh 
python app.py
```
